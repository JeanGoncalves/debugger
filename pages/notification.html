Notificações

Info

O console.info exibe uma mensagem de informação no console.

Exemplo: 
                    
    var name = 'User';
    console.info('Autenticação completa. Usuário: %s',name  );

Warn

O console.warn exibe uma mensagem de alerta no console.

Exemplo: 
                    
    var num = 10;                
    console.warn('Número inesperado: $i', num);

Error

O console.error exibe a mensagem em formato de erro na notificação.

Exemplo: 
                    
    function connectToServer() {    
        var errorCode = 1;    
        if (errorCode) {        
            console.error("Error: %s (%i)", "Server is  not responding", 500);    
        }
    }
    connectToServer();
