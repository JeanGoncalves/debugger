Profile

O console.profile identifica quando inicia e finaliza a chamada do debug.

Exemplo: 
                
function processPixels() {
    console.profile("Processing pixels");  
    console.profileEnd();
}
processPixels();
// Exibe:
// Profile 'Processing pixels' started.
// Profile 'Processing pixels' finished.

https://developer.chrome.com/devtools/docs/console-api#consolecountlabel
