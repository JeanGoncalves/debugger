Debugger

O debugger insere um ponto de parada na execução do código javascript, então apertando F11 pode ir vendo passo-a-passo a funcionalidade doo script.

                
    var brightness = function() {
        debugger;

        var x = Math.floor((Math.random() * 100) + 1);
        var y = Math.floor((Math.random() * 100) + 1);
        var z = Math.floor((Math.random() * 100) + 1);

        var response = (x+y+z);
        return response;

    };

    var b = brightness();
