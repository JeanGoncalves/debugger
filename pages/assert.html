Assert

O console.assert trabalha com condição, quando essa condição retornar FALSE ele imprime o retorno.

Exemplo:

var list = document.getElementById('#myList');
var cond = list.length < 3;
var retorno = "A quantidade da lista é menor que 3";

console.assert(cond, retorno);
